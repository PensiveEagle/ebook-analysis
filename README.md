## About The Project

A small project to analyse the e-book "Miracle in the Andes" using Python in Jupyter notebooks.

The notebook starts with practice with regular expressions to extract specified parts of the book.

The second part of the notebook uses Natural Language Processsing with the nltk library to perform sentiment analysis on each of the chapters to determine which chapter is the most positive and which is the most negative.
